Module FilterGraph

Filter graph drawing module.

Subclass of Graph for drawing common filter graphs. Can draw approximations of low-pass, band-pass, notch and high-pass filters.

Info:

  • Release: v1.0.0
  • Author: Mark Eats

Functions

new (x_min, x_max, y_min, y_max, filter_type, slope, freq, resonance)
Create a new FilterGraph object. All arguments optional.

Parameters:

  • x_min Minimum frequency value in Hz for x axis, defaults to 20.
  • x_max Maximum frequency value in Hz for x axis, defaults to 24000.
  • y_min Minimum amplitude value in dB for y axis, defaults to -60.
  • y_max Maximum amplitude value in dB for y axis, defaults to 30.
  • filter_type Type of filter, accepts "lowpass", "bandpass", "notch" or "highpass", defaults to "lowpass".
  • slope Slope value in decibels per octave, defaults to 12.
  • freq Frequency value in Hz, defaults to 2000.
  • resonance Resonance value 0-1, defaults to 0.

Returns:

    Instance of FilterGraph.
